# Smart Profit & Loss Calculator

A Java-based Smart Profit & Loss Calculator built using OOP, AWT/Swing, and exception handling. It calculates profit or loss based on cost price, selling price, quantity, and tax/discount inputs. Includes a user-friendly GUI, error validation, and clear results.

## Features
- Interactive GUI for easy use
- Calculates Profit/Loss Amount and Percentage
- Supports Quantity, Tax, and Discount
- Error handling for invalid inputs
- Built with OOP (Classes, Methods, Constructors)
